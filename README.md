💈 BarberApp

Sistema completo para gerenciamento de barbearias — moderno, rápido e totalmente integrado.

Criado para facilitar o dia a dia de clientes, barbeiros e administradores em uma única aplicação fullstack.

🚀 Sobre o Projeto

O BarberApp oferece uma experiência totalmente digital para qualquer barbearia.

Clientes realizam agendamentos com poucos cliques, enquanto administradores possuem um painel completo para controlar serviços, profissionais, produtos, financeiro e toda a rotina diária.

A aplicação foi desenvolvida utilizando uma arquitetura fullstack moderna com:

Frontend e Backend integrados no próprio Next.js
API Routes do Next.js
Prisma ORM
MongoDB
Autenticação segura
Interface moderna e responsiva
